(4)Canvas and SVG tags.

Ans:Canvas tag is used to draw graphics, via JavaScript. The canvas element is a container for graphics.Scalable Vector Graphics useful for defining graphics such as boxes, circles, text, etc. and is a language for describing 2D-graphics.
